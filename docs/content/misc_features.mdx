title: Miscellaneous Features

This page lists features and changes that are either not effects or aren't sound/particle effects

## Particle Culling
- Off screen particles are culled
- Particles now have a render distance config that will prevent rendering particles a certain number of chunks away
  (defaults to 5). Certain particles will ignore this config depending on how they are spawned in the world such as
  campfire smoke
- Particles in unloaded chunks are culled

## Mob Skull Shaders
Wearing a mob skull on your head will apply the [Spectator Mob View](@minecraft:Spectator#Mob_view) shader.
By default, this effect is only applied to Creeper and Ender Dragon heads, however any helmet equipment can have a
shader applied to it with a resource pack. See [Mob Skull Shaders]($documentation/mob_skull_shaders)

## Changes to Name Tag Rendering
When a mob is sleeping in a bed the name tag is repositioned to be at the top of the head, rather than floating above
the center of the mob allowing for better visibility of snoring particles. This applies to all mobs, but is mainly
noticeable for players and villagers

## Biome Color Water Particles Resource Pack
A builtin resource pack that changes the color of all vanilla water related particles to match the water color of the
biome they are spawned in.

## Biome Color Rain
Changes the rain color to match the water color of the biome the rain is in. This is fully independent of
the [Biome Color Water Particles Resource Pack](#biome-color-water-particles-resource-pack).
If you are experiencing issues with rain while using a shader, try disabling this feature.
